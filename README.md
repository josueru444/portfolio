## Tecnologías

Este proyecto está construido con un stack moderno enfocado en rendimiento y diseño:

-   **Core:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
-   **Estilos:** [Tailwind CSS v4](https://tailwindcss.com/)
-   **Iconos:** [Lucide React](https://lucide.dev/)
-   **Animaciones:** CSS nativo (Keyframes) + Intersection Observer API
-   **Despliegue:** GitHub Pages / Vercel

##  Características

-   **Diseño "Soft Solid":** Interfaz oscura, elegante, con tarjetas semitransparentes y desenfoques (backdrop-blur).
-   **Cursor Personalizado:** Implementación propia de un cursor minimalista con físicas de seguimiento y estados reactivos.
-   **Animaciones al Scroll:** Los elementos aparecen suavemente mientras navegas (Scroll Reveal).
-   **Performance:** Optimizado con Code Splitting y carga diferida (Lazy Loading) de componentes.
-   **Responsive:** Totalmente adaptado a móviles, tablets y escritorio.
-   **CV Interactivo:** Visualizador de PDF integrado en un modal sin salir de la página.

##  Instalación y Uso Local

Si deseas clonar y correr este proyecto en tu máquina local:

1.  **Clonar el repositorio**
    ```bash
    git clone https://github.com/josueru444/portfolio.git
    cd portfolio
    ```

2.  **Instalar dependencias**
    ```bash
    npm install
    ```

3.  **Correr servidor de desarrollo**
    ```bash
    npm run dev
    ```
    El sitio estará disponible en `http://localhost:5173`.

##  Despliegue

### GitHub Pages

El proyecto está configurado para desplegarse automáticamente en GitHub Pages usando `gh-pages`.

```bash
npm run deploy
```

### Vercel

Configurado con `vercel.json` para soportar rutas SPA. Simplemente conecta el repositorio a Vercel y detectará la configuración de Vite automáticamente.
